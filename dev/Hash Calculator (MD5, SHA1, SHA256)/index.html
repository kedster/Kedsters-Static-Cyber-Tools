What are Hash Functions?

Cryptographic hash functions are mathematical algorithms that convert input data into a fixed-size string of bytes. The output is deterministic and appears random.

Common Uses

  • File integrity verification
  • Password storage (with salt)
  • Digital signatures
  • Data deduplication
  • Proof of work (blockchain)

Security Properties

  • One-way: Irreversible computation
  • Deterministic: Same input = same output
  • Avalanche effect: Small change = big difference
  • Collision resistance: Hard to find duplicates
