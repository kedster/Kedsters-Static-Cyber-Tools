What is URL Encoding?

URL encoding (percent-encoding) is a mechanism to encode special characters in URLs by replacing them with a percent sign followed by their hexadecimal representation.

Common Use Cases

  • Safely transmitting data in URLs
  • Handling special characters in query strings
  • Web security testing and analysis
  • API parameter encoding
  • Form data submission

Security Considerations

  • Prevent URL injection attacks
  • Double encoding can bypass filters
  • Always validate decoded URLs
  • Watch for malicious redirects
  • Check for homograph attacks
